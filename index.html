Redirection www

www.ecole-gustave.com/* → ecole-gustave.com/ (perte du chemin)
Critique
Quand on visite www.ecole-gustave.com/formations/, la redirection 301 envoie vers ecole-gustave.com/ au lieu de ecole-gustave.com/formations/.
Impact : Tous les backlinks pointant vers des pages en www perdent leur valeur SEO. Les moteurs de recherche traitent www et non-www comme des sites différents.
Solution : Corriger la règle de redirection (Nginx/Apache/Cloudflare) pour préserver le chemin : www → non-www + même URI.

URLs staging en production

12 ressources chargées depuis le staging Infomaniak
Critique
Le domaine 1a13o2bxfri.preview.infomaniak.website est utilisé pour charger logos et icônes en production.
Impact : Si le staging est supprimé, 12 images deviennent des 404. Casse l'image de marque.
Solution : Remplacer dans la base WordPress (wp_posts.post_content) le staging par ecole-gustave.com via SQL UPDATE.

Détail des problèmes de sécurité

Conformité RGPD

Facebook Pixel chargé avant consentement
Critique RGPD
Le pixel Facebook (facebook.com/tr?id=484946844150303) est chargé directement dans le <head>, avant que le bandeau cookies Complianz n'ait recueilli le consentement.
Non-conforme au RGPD et à la directive ePrivacy. Risque d'amende CNIL (amendes récentes : 40M€ pour Google, 60M€ pour Amazon).
Configurer Complianz pour bloquer le pixel Facebook jusqu'au consentement explicite de l'utilisateur.

JSON-LD (données structurées)

Aucun JSON-LD sur la page d'accueil
Haute priorité
Google utilise les données structurées pour générer des rich snippets (étoiles, breadcrumbs, infos entreprise). Actuellement : aucun JSON-LD présent.
Types recommandés : Organization, LocalBusiness, BreadcrumbList, Course (pour les formations).
Activer le schema JSON-LD dans Yoast SEO (Organization + Breadcrumb). Ajouter manuellement les schémas Course pour les formations.
